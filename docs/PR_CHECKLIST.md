# Pull Request Checklist

- [ ] `npm install` succeeds
- [ ] `npm run build` succeeds
- [ ] Upload works with PNG
- [ ] Upload works with SVG
- [ ] Generated ZIP contains favicon, apple, android, pwa, microsoft, social, and generic folders
- [ ] `manifest.webmanifest` is generated in the ZIP
- [ ] `browserconfig.xml` is generated in the ZIP
- [ ] Quick Editor can update a generated PNG and refresh the ZIP
- [ ] Console has no warnings or errors during the main flow
- [ ] Layout is usable on desktop and mobile widths
